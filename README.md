 ## Tech Stack
  ### Front End:
    1. React 
    2. Axios for api calls
    3. React-router-dom for page navigation
  ### Backend:
    1. Nodejs
    2. Express Server
    3. Mongoose ORM
    4. MongoDB

To run, run npm install to download the node_modules
